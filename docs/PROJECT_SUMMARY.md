### 2. MVP原型系统

```
src/
├── app.py                      # Streamlit主应用
├── models/                     # 数据模型
│   ├── radar.py               # 雷达配置
│   ├── turbine.py             # 风机配置
│   ├── target.py              # 目标配置
│   ├── scene.py               # 场景配置
│   └── results.py             # 结果模型
├── engine/                     # 评估引擎
│   ├── blocking.py            # 遮挡模型
│   ├── scattering.py          # 散射模型

│   ├── accuracy.py            # 精度模型
│   └── eval_engine.py         # 评估引擎主类
└── utils/                      # 工具函数
    ├── geo_utils.py           # 地理计算
    └── file_utils.py          # 文件操作
```

### 技术栈

- **Web框架**: Streamlit 1.28+
- **可视化**: Plotly 5.18+ (图表), Streamlit Map (地图)
- **数值计算**: NumPy 1.24+, SciPy 1.11+
- **数据处理**: Pandas 2.0+

## 系统特点

### 1. 模块化设计
- 数据模型、评估引擎、可视化分离
- 便于维护和扩展

### 2. 可视化丰富
- 地图展示（雷达站、风机位置）
- 图表分析（时域、频域、距离剖面）
- 雷达图（综合评估）

### 3. 预设库支持
- 风机型号库（Vestas、GE、金风等）
- 雷达波段自动识别
- 目标类型库（民航、战机、无人机等）

### 4. 评估报告
- 综合风险等级判定
- 各项评估指标详细展示
- 智能建议生成

## 使用说明

### 启动应用

```bash
# Linux/Mac
./run.sh

# Windows
run.bat
```

或手动启动：

```bash
cd src
streamlit run app.py
```

### 使用流程

1. **配置雷达参数** - 在左侧边栏设置雷达工作参数
2. **添加风机** - 选择型号、设置位置偏移、点击添加
3. **配置目标** - 选择目标类型或自定义参数
4. **执行评估** - 点击"开始评估"按钮
5. **查看结果** - 在四个标签页查看详细评估结果
