## 1. 引言

### 1.1 编写目的

本文档旨在明确风机-雷达干扰评估系统的功能需求、性能需求、接口需求及约束条件，为系统设计、开发、测试和验收提供依据。

### 1.2 项目背景

随着风电产业的快速发展，大型风电场与雷达站之间的电磁兼容性问题日益突出。风机的旋转叶片会对雷达探测产生遮挡、散射、多普勒频移等干扰效应，影响雷达对目标的正常探测和跟踪。本系统旨在建立科学的评估模型，量化分析风机对雷达探测能力的影响，为风电场选址和雷达站规划提供决策支持。

### 1.3 术语定义

| 术语 | 定义 |
|------|------|
| RCS | 雷达散射截面 (Radar Cross Section) |
| 多普勒频移 | 由于相对运动导致的雷达回波频率变化 |
| 遮挡因子 | 风机对雷达信号路径的物理阻挡程度 |
| 散射系数 | 风机结构对雷达信号的散射能力 |
| 绕射损耗 | 信号绕过障碍物传播时的能量损失 |
| 测角精度 | 雷达测量目标方位角的准确程度 |

### 1.4 参考资料

1. ITU-R M.1849-1 风电场对雷达系统影响的评估方法
2. IEEE Std 149-2021 雷达散射截面测量标准
3. GB/T XXXXX 风电场电磁兼容性评估规范

## 2. 任务概述

### 2.1 目标

开发一套专业的风机-雷达干扰评估系统，实现：
- 单/多风机对雷达探测能力影响的定量评估
- 三维场景可视化展示
- 多维度评估报告生成
- 评估结果的空间分布分析

### 2.2 用户特点

| 用户类型 | 技术背景 | 使用场景 |
|----------|----------|----------|
| 雷达工程师 | 电磁兼容专业 | 雷达站选址评估 |
| 风电规划师 | 风电工程专业 | 风电场合规性评估 |
| 空域管理人员 | 航空管制专业 | 空域安全评估 |
| 科研人员 | 电磁场/信号处理 | 学术研究分析 |

- **客户端**: Chrome/Firefox/Edge 最新版本
- **硬件要求**: 支持WebGL的显卡（3D可视化）

### 3.3 评估模型详细说明

#### 3.3.1 遮挡模型

```
遮挡因子计算：
η_block = (A_turbine ∩ A_beam) / A_beam × 100%

其中：
- A_turbine: 风机在雷达视线方向的投影面积
- A_beam: 雷达波束在风机距离处的截面面积

遮挡持续时间：
T_block = (N_blade × θ_blade) / (2π × RPM/60) × 100%

其中：
- N_blade: 叶片数量
- θ_blade: 叶片在视线方向的角宽度
- RPM: 风机转速
```

#### 3.3.2 散射模型

```
散射功率计算（雷达方程）：
P_r = (P_t × G_t × G_r × λ² × σ) / ((4π)³ × R⁴ × L)

其中：
- P_t: 发射功率
- G_t, G_r: 发射/接收天线增益
- λ: 波长
- σ: 风机RCS
- R: 雷达-风机距离
- L: 系统损耗

信干比：
SJR = P_target / P_turbine
```

#### 3.3.3 绕射模型

```
基于一致性绕射理论(UTD)：
L_diffract = 20 × log₁₀|D(θ) × √(λ/(2πd)) × e^(-jkd)|

其中：
- D(θ): 绕射系数
- θ: 入射角
- d: 绕射点到接收点距离
- k: 波数
```

#### 3.3.4 多普勒模型

```
叶片尖端多普勒频移：
f_d = (2 × v_tip × f₀) / c

其中：
v_tip = 2π × R_blade × RPM / 60
- R_blade: 叶片长度
- f₀: 雷达工作频率
- c: 光速

多普勒谱宽：
Δf = (2 × (v_tip - v_root) × f₀) / c
```

#### 3.3.5 测角偏差模型

```
角度测量误差：
Δθ = θ_3dB × √(P_turbine / P_target) × sin(φ)

其中：
- θ_3dB: 天线3dB波束宽度
- φ: 干扰信号与目标信号的相位差
```

#### 3.3.6 测距/测速偏差模型

```
距离测量误差（多径效应）：
ΔR = (c × Δτ) / 2
其中 Δτ = τ_direct - τ_multipath

速度测量误差：
Δv = (λ × Δf_d) / 2
```

## 4. 性能需求

### 4.1 精度要求

| 评估指标 | 精度要求 | 说明 |
|----------|----------|------|
| 遮挡因子 | ±2% | 基于几何光学计算 |
| 散射功率 | ±3dB | 考虑RCS测量误差 |
| 多普勒频移 | ±10Hz | 基于转速精度 |
| 测角偏差 | ±0.1° | 基于信干比 |
| 测距偏差 | ±10m | 基于多径模型 |
| 测速偏差 | ±1m/s | 基于多普勒精度 |

### 4.2 响应时间

| 操作类型 | 响应时间要求 | 说明 |
|----------|--------------|------|
| 地图加载 | ≤3秒 | 初始地图渲染 |
| 单风机评估 | ≤1秒 | 单点计算 |
| 多风机评估(≤50) | ≤5秒 | 批量计算 |
| 多风机评估(≤500) | ≤30秒 | 大规模风电场 |
| 3D场景渲染 | ≤2秒 | 三维可视化 |
| 报告生成 | ≤10秒 | PDF报告导出 |

### 4.3 并发要求

- 单用户多场景并发：支持同时打开5个场景标签
- 多用户支持：基于Streamlit的会话隔离

### 4.4 数据容量

| 数据类型 | 容量要求 | 说明 |
|----------|----------|------|
| 场景配置 | 无限制 | 本地JSON存储 |
| 风机数据 | ≤10000条 | CSV/Excel导入 |
| 评估结果 | 无限制 | 实时计算不存储 |
| 历史报告 | ≤1000份 | 本地存储 |
