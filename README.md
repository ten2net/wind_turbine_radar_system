# 风机-雷达干扰评估系统

## 项目简介

风机-雷达干扰评估系统（Wind Turbine - Radar Interference Assessment System）是一款专业的电磁兼容性评估工具，用于量化分析风电场对雷达探测能力的影响。

## 核心功能

### 评估指标

| 指标类别 | 具体指标 | 说明 |
|----------|----------|------|
| 遮挡效应 | 遮挡因子 | 风机对雷达信号路径的物理阻挡比例 |
| 遮挡效应 | 遮挡持续时间 | 叶片旋转导致的遮挡时间占比 |
| 散射效应 | 干扰功率 | 风机散射的雷达信号功率 |
| 散射效应 | 信干比(SJR) | 目标回波与干扰信号功率比 |
| 多普勒效应 | 多普勒频移 | 旋转叶片产生的频谱污染 |
| 多普勒效应 | 多普勒带宽 | 叶片速度分布导致的谱展宽 |
| 精度偏差 | 测角偏差 | 散射信号导致的角度测量误差 |
| 精度偏差 | 测距偏差 | 多径效应导致的距离测量误差 |
| 精度偏差 | 测速偏差 | 多普勒污染导致的速度测量误差 |

### 技术栈

- **前端框架**: Streamlit
- **地图服务**: Streamlit内置地图 + PyDeck
- **图表库**: Plotly
- **数值计算**: NumPy, SciPy
- **数据处理**: Pandas

## 快速开始

### 环境要求

- Python 3.10+
- pip 或 conda

### 安装步骤

1. 克隆仓库
```bash
cd wind_turbine_radar_system
```

2. 创建虚拟环境
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# 或
venv\Scripts\activate  # Windows
```

3. 安装依赖
```bash
pip install -r requirements.txt
```

4. 运行应用
```bash
cd src
streamlit run app.py
```

5. 在浏览器中打开 http://localhost:8501

## 使用指南

### 1. 配置雷达参数

在左侧边栏中设置雷达的工作参数：
- 工作频率 (GHz)
- 发射功率 (kW)
- 天线增益 (dBi)
- 波束宽度 (度)
- 天线高度 (m)
- 最大探测距离 (km)

### 2. 添加风机

1. 选择预设风机型号或自定义参数
2. 设置纬度/经度偏移（相对于雷达位置）
3. 点击"添加风机"按钮

### 3. 配置目标参数

选择目标类型（民航客机、战斗机、无人机等），系统会自动加载典型参数。

### 4. 执行评估

点击"开始评估"按钮，系统将计算各项干扰指标。

### 5. 查看结果

评估结果分为四个标签页：
- **遮挡分析**: 显示遮挡因子、持续时间、时域变化
- **散射分析**: 显示干扰功率、信干比、距离剖面
- **多普勒分析**: 显示多普勒频移、频谱特征
- **精度分析**: 显示测角/测距/测速偏差

## 算法模型

### 遮挡模型

基于几何光学原理计算风机的投影面积与雷达波束截面积的比例：

```
遮挡因子 = (风机投影面积 / 雷达波束截面积) × 100%
```

### 散射模型

基于雷达方程计算风机散射功率：

```
P_r = (P_t × G² × λ² × σ) / ((4π)³ × R⁴ × L)
```

### 多普勒模型

计算旋转叶片产生的多普勒频移：

```
f_d = (2 × v_tip × f₀) / c
```

### 精度模型

分析散射干扰对测量精度的影响：

```
测角偏差 = 波束宽度 × √(干扰功率/目标功率)
```
